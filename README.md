# Cheats

Today's software engineers are required to be familiar with multiple programming languages, the APIs of a variety of application libraries and frameworks, and the commands of numerous software development tools. On top of all that, we'll typically dabble in systems administration and other concerns that sit adjacent to our area of expertise.

No software engineer can possibly memorise all the APIs that we are required to use in our professional work.

Cheat sheets provide a handy quick reference to the terms, commands and symbols that we need to understand but not necessarily memorise. These are my personal cheat sheets for the languages and tools, and recurring manual processes, that I use again and again.
